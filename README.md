# NLP-Sentiment-Analysis-of-Free-Responses
Notebooks and scripts used to analyze sentiment analysis with and without POS tagging for short narrative stories containing different levels of disfluency

The purpose of these analyses are to to analyze the relationship between speaker impressions and number of filled pauses in the passage.

For access to the deidentified data, please reach out to Caitlin Volante at caitlin.r.volante@vanderbilt.edu

## Sentiment Analysis without POS Tagging

Data cleaning and linear mixed effects models for the positve/negative sentiment analysis without POS tagging can be found in the NLP_SENTIMENT.Rmd file 

Data cleaning, tag removal, sentiment analysis, and graph creation can be found in the Word_Properties_Assignment.ipynb

## Sentiment Analysis with POS Tagging

Data cleaning and linear mixed effects models for the positve/negative sentiment analysis without POS tagging can be found in the NLP_POS_SENTIMENT.Rmd file.

Data cleaning, tag removal, POS tagging, sentiment analysis, and graph creation can be found in the  POS_Assignment.ipynb
